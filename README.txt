Для начала работы необходимо подключить файл OurStructs.h

#include "OurStructs.h"

Файлы dint.c и lists.c должны находится в одной папке с OurStructs.h

Документация содержится в коде.

Файл dint.c содержит в себе структуру динамического массива и матрицы для динамических массивов целочисленных значений, а так же набор функций для работы с ними.
Файл lists.c содержит в себе структуру список и стэк целочисленных значений, а так же набор функций для работы с ними.
